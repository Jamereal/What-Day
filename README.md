A focused, minimal daily task companion web app designed for iPhone.
Features a cute minimal pink theme, Siri voice reading, and local notifications.

## Features
- **Daily Events**: Add, edit, and manage daily tasks.
- **Voice Reading**: Tap the button to hear your daily summary (uses Siri voice on iOS).
- **Notifications**: Get notified 15 mins before tasks and a morning summary.
- **Cute Theme**: Minimal white & hot pink design.
- **PWA**: Installable on iPhone Home Screen.
- **Offline Capable**: Works without internet.

## How to install on iPhone
1. Open the website in Safari.
2. Tap the "Share" button.
3. Scroll down and tap "Add to Home Screen".
